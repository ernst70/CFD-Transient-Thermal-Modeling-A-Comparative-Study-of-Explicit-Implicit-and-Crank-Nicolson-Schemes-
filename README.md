# CFD-Transient-Thermal-Modeling-A-Comparative-Study-of-Explicit-Implicit-and-Crank-Nicolson-Schemes-
This project focused on the development of a transient, two-dimensional (2D r-z) cylindrical CFD solver from scratch in MATLAB to analyze the unsteady-state thermal dynamics inside a single catalyst pellet. Formulated around the 2D unsteady heat conduction equation in cylindrical coordinates.
